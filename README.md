## 项目运行
***
通过npm安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))
#### 注意：由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本 
```
cd vue-shop

npm install 或 cnpm install(个人比较喜欢使用后者，下载依赖模块速度较快)

npm run dev

```
